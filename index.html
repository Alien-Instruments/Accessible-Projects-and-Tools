Visual Impairment Filters and Colour Contrast Checker

Building accessible software and websites is crucial to ensuring that everyone, including visually impaired and blind users, can engage with digital content. This tool is designed to help developers and designers create more inclusive products. By allowing users to upload images and apply visual impairment filters, it offers a unique perspective on how designs appear to individuals with various visual impairments. Additionally, the built-in WCAG contrast checker ensures that your designs meet essential accessibility standards for text and background contrast. With this tool, you can easily identify and address potential accessibility issues early in the design process.

Alien Instruments UBIK: The Talking MIDI Mapper

Welcome to Alien Instruments UBIK, the talking MIDI mapping solution. Create groups for your controls and start adding sliders and buttons to send MIDI CC’s. Completely accessible for screen reading and keyboard navigation. Labelling groups and components also labels for screen reading. Design your own interface and map it to the device or software you wish to control. Each slider has a MIDI learn button, click this and follow the voice prompts to map your controller. When adding a slider, you can map the output range displayed on the screen and spoken aloud to and integer or float to match the parameter you are controlling. Alternatively, you can display or hear the current MIDI CC value. For buttons you can set the on value to any cc value, when mapped this will speak as “parameter name” on and off and display the same in the button. To get started click the add group button and follow the prompts. Or take a guided tour of the interface and settings by clicking the "Start Tour" button.

If you are using a screen reader to take the tour after clicking start tour focus will go to a heading level 2, this is the parameter name, and that component shows focus in the UI. Use the tab key and next will be a heading level 3 with a description and context of the area or item. Press tab again and you will focus on the next button, followed by previous and then close buttons. When the tour ends there will be a finish button.

Have fun and enjoy!

Alien Instruments Groove Table

This project is a tribute to two iconic instruments that shaped the sound of electronic music: the Roland TR-808 drum machine and the TB-303 bass synthesizer. Built entirely with modern web technologies using the Web Audio API and Web MIDI, this application reimagines these classic machines in a browser environment—no plugins or downloads required.

The project features two fully functional synthesizer interfaces:

A skeuomorphic mode that visually mimics the original hardware layout and styling for nostalgic fidelity

A custom UI mode offering a modern, streamlined interface optimized for performance and accessibility

Both interfaces are fully accessible, designed with inclusive practices in mind. Screen reader support, keyboard navigation, and responsive layouts ensure usability across a wide range of devices and user needs. Under the hood, synchronized audio engines drive the 808-inspired drum sequencer and 303-style bassline generator, with tempo sync, swing, accent, and parameter automation all supported.

This homage aims not only to preserve the spirit of these instruments but also to invite new experimentation through the flexibility of the web.

Alien Instruments Web Audio Groovebox

Welcome to the Web Audio Groovebox — a fully browser-based music production environment powered by the Web Audio API. This project brings together a suite of integrated tools for electronic music creation, all running natively in your web browser with no plugins or downloads required.

Whether you're laying down beats, programming basslines, slicing samples, or crafting polyphonic melodies, the Groovebox provides intuitive, hands-on control through a clean and responsive UI.

Features

Drum Machine: Multi-voice drum synthesizer with individual step sequencers, pitch/decay/level controls, and accent toggling.

Bass Synth (TB-303 style): Acid-style monosynth with slide, tie, and accent sequencing, plus full control over filter, envelope, and distortion parameters.

Sampler: Load your own audio files or factory samples, with control over start/end points, reverse playback, envelope shaping, and filter settings.

Polyphonic Synth: Two-oscillator poly synth with filter and amp envelopes, per-step sequencing, and chord playback support.
